# GreenTensor Web — рассеяние на многослойной сфере

Локальное веб-приложение поверх канонического ядра слоистой сферы
`green_tensor/01_sphere.py` (импорт через фасад `green_tensor/sphere_core.py`;
сверено с аналитическим рядом Ми, см. `tests/run_all.py`).

## Запуск

```bash
python3 webapp/server.py            # http://127.0.0.1:8765
python3 webapp/server.py --port 9000
```

Зависимости — только те, что уже нужны ядру: `numpy`, `scipy` (веб-сервер — stdlib,
plotly.js лежит локально в `static/vendor/`, интернет не требуется).

## Вход

* **Слои**: радиус `r` (нормируются к внешнему автоматически), `ε′`, `ε″`
  (Im ε > 0 — поглощение, конвенция e^{−iωt}; большие значения ~1e7 — модель металла/PEC),
  `μ′`. Внешний слой-воздух добавляется автоматически.
* **k₀·R** — волновой размер на внешнем радиусе; **toch** — число гармоник
  (пусто = авто по правилу Уискомба); **φ** — азимут наблюдения для линейной ДН.
* **Задача**: дифракция плоской волны и/или «антенна» (источник на поверхности сферы).
* **Свип по k₀** — диапазон и число точек для кривых сечений.
* Пресеты: Люнеберг (7 и 4 слоя), металлическая сфера, металл с укрытием, однородный диэлектрик.

## Выход (все диаграммы в едином стиле)

| Блок | Содержимое |
|------|------------|
| Геометрия | сечение слоёв с подписями ε |
| Карточки | Q_sca, Q_ext, Q_abs, Q_back при текущем k₀ |
| Линейная поляризация | главные плоскости — E-плоскость (φ=0°) и H-плоскость (φ=90°), как в авторских сравнениях с HFSS; дополнительный срез φ (пунктир, E_θ и E_φ). В главных плоскостях E_φ ≡ 0 — это свойство оригинальной формулы (E_φ ∝ sin φ·cos φ) |
| Круговая поляризация | основная (co) и кросс-компоненты, полярная + развёртка, дБ |
| Сечения | Q(k₀) — полное/экстинкция/поглощение/обратное, маркер текущего k₀ |
| Коэффициенты | \|M_n\|, \|N_n\| по гармоникам (лог-шкала) |

дБ-нормировка — совместная для пары кривых (к общему максимуму), видны относительные
уровни кросс-поляризации. Экспорт: PNG (кнопка-камера на каждом графике),
JSON (полный ответ), CSV (все диаграммы).

## API

`POST /api/compute` — JSON:

```json
{
  "k0": 10, "toch": null, "phi_deg": 45,
  "layers": [{"r": 0.5, "eps_re": 2.25, "eps_im": 0, "mu_re": 1}, {"r": 1.0, "eps_re": 1.5}],
  "problems": ["diffraction", "antenna"],
  "sweep": {"enable": true, "k0_min": 0.25, "k0_max": 20, "points": 160}
}
```

Ответ: `theta_deg`, `results.{diffraction,antenna}.{linear,circular,coeffs,cross_sections}`,
`sweep`, `meta` (использованный toch, нормированные слои, предупреждения).
