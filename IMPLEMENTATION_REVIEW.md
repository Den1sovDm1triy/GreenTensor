# Ревью реализации против теории (`GreenTensor_Theory.tex`)

Сверка кода (`green_tensor/`) с теоретическим документом: конвенции, формулы, покрытие,
расхождения. Каждая помеченная нестыковка проверена независимым агентом-скептиком.

## Сводка

| Область | Вердикт | Реальные расхождения |
|---|---|---|
| Слоистая сфера (`mie_core.py`, `01_sphere.py`) | ✅ consistent | нет |
| T-матрица (`tmatrix.py`, `scatterer.py`) | 🟡 minor_gaps | нет (footgun-именование, задокументировано) |
| ВСВФ + трансляция (`vswf.py`) | ✅ consistent | нет |
| GMM (`gmm.py`) | 🟡 minor_gaps | нет (опровергнуто) |
| Эллипсоид/сфероид (`ellipsoid.py`, `spheroid.py`) | ✅ consistent | нет |
| Цилиндр/конус (`cylinder.py`, `cone.py`) | 🟡 minor_gaps | нет (документированный объём) |
| Разложение (`decompose.py`) | ✅ consistent | нет |

**Итог: 0 реальных расхождений код↔теория.** Конвенции (`e^{-iωt}`, исходящая `h_n^{(1)}`,
`c=Td`, `S=I+2T`) согласованы во всех модулях. Оба изначально помеченных «mismatch» при
адверсариальной проверке понижены (см. ниже).

## Ключевые подтверждения

- **Сфера**: Риккати–Бессель `ψ/χ/ξ`, производные-рекурренции, импедансно-адмитансная
  рекурсия Адена–Керкера (Мёбиус), коэффициенты `Mn/Nn`, диаграмма `S1/S2`, сечения
  `Q_sca/Q_ext/Q_abs/Q_back` — **все совпадают** с уравнениями теории. Знаковые «причуды»
  (χ без знака; сопряжение `Mn,Nn`; общий `(-1)^1` в угловых) задокументированы в теории и
  сокращаются в наблюдаемых. Энергобаланс выполняется в коде.
- **ВСВФ**: `M=∇×(rψ)`, `N=(1/k)∇×M`, скалярная трансляция `α` (тождество полей ~1e-16),
  векторная `A,B` — структура совпадает; используется `h_n^{(1)}`, фаза Кондона–Шортли.
- **GMM**: самосогласованная система `a_p=d_p+Σ G T a_q`, `c_p=T a_p`, разложение плоской
  волны, сечения — соответствуют теории.
- **Эллипсоид/сфероид**: факторы деполяризации (Σ=1), поляризуемость (однородный +
  конфокальное покрытие), рэлеевские сечения — соответствуют квазистатическому разделу.

## Разбор помеченных нестыковок (обе — НЕ дефект)

1. **GMM C_ext (опровергнуто).** Код `gmm.py` использует физическую амплитуду рассеяния
   `S` и оптическую теорему `C_ext=(4π/k)·Im[ê*·S]` (= `decomp-eq:optical-theorem`), а не
   гармоническую форму `(4π/k²)·Re[ê*·F]` (`tm-Cext`), где `F=k·S`. Формы эквивалентны;
   код физически корректен (проходит lossless-энергобаланс `C_abs≈0`). Сравнение с `tm-Cext`
   было категориальной ошибкой.
2. **T-матрица M/N-маппинг (partial).** Численно всё верно и **задокументировано в теории**:
   в `mie_core` `Mn` (из импеданса `Z`) — это электрический `a_n` (TM), `Nn` (из адмитанса
   `Y`) — магнитный `b_n` (TE). Поэтому `t_N=−Mn (=−a_n)`, `t_M=−Nn (=−b_n)`. Риск — только
   именование для будущих потребителей GMM.

## Задокументированные сокращения объёма (не дефекты)

- Полноволновой сфероид, конечный цилиндр/конус, прямой сферо-конический решатель → честный
  `NotImplementedError` (нужны спецфункции с комплексным `c` / нецелые Лежандра /
  мод-матчинг). Сложные тела решаются через `decompose → GMM`.
- Векторная трансляция `A,B` — спектральная проекция; точна при `k·разнос ≳ nmax`, для
  плотных упаковок нужны замкнутые коэффициенты Cruzan (страж предупреждает).
- GMM сейчас связывает диагональные T (сферы); полная недиагональная кластерная T и
  повороты Вигнера-D — следующий слой.
- Эллипсоид: конфокальное покрытие реализовано для 2 слоёв (не произвольного N).

## Внесённые по итогам ревью правки

- `tmatrix.from_mie_coeffs`: явное пояснение электр./магн. маппинга (`Mn=a_n`, `Nn=b_n`).
- `gmm.extinction_cross_section`: док-примечание о физической форме оптической теоремы.
- `ellipsoid.depolarization_factors`: подавлено шумное `IntegrationWarning` (результат точен).
- `ellipsoid.orientation_average_cross_sections`: добавлен хелпер `⟨C⟩=(1/3)ΣC_j`
  (theory eq:orientation-average), с тестом.
- Полный набор тестов: **54/54 зелёные**.
