# Верификация решателя сферы (`green_tensor/01_sphere.py`)

Расчёт рассеяния на многослойной сфере сверен с **независимым аналитическим рядом Ми**
(Bohren & Huffman) — `tests/analytic_mie.py`. Эталон самопроверяется (релеевский предел
рассеяния и поглощения, парадокс экстинкции для PEC) и не зависит от кода библиотеки.

Запуск всех проверок (pytest не требуется):

```bash
python3 tests/run_all.py        # или: pytest tests/
```

## Результат: 10/10 проверок пройдено

| Режим | Что сверяется | Точность |
|------|----------------|----------|
| Однородная сфера | Q_sca vs Ми, x=0.5…5 | машинная (~1e-15) |
| Обратное рассеяние | Q_b vs Ми (чувствительно к знаку) | ~1e-15 |
| Многослойность | инвариантность к разбиению на N идентичных слоёв | ~1e-15 |
| Разные ε | «вакуумная оболочка» = меньшая сфера | ~1e-15 |
| **Поглощение** | Q_sca и Q_abs vs Ми, loss=0.01…5 | ~1e-15 |
| **Знак поглощения** | Im(ε)>0 → Q_abs>0; Im(ε)<0 → усиление | ✓ |
| **Реальный металл** | Re(ε)<0 vs точная mie_ab(√ε) | ~1e-5 |
| **PEC-предел** | \|ε\|→∞ (−1.7e7j) конечен и → mie_pec | 0.1% |
| **Металл-диэлектрик** | металлическая оболочка экранирует ядро → PEC | 0.0% |

## Найденные и исправленные баги

Все правки подтверждены арбитром (числа меняются в сторону аналитики; lossless-регрессия цела).

1. **C4 — `eps.append(len(eps))`** (был self-terminating хак): во внешний слой
   дописывался счётчик-сентинел вместо воздуха ε=1; рекурсия импедансов на внешней
   границе портилась. Эффект зависел от числа слоёв (до 35% при малых ka).
   → Внешняя среда = воздух (ε=1), дописывается один раз.

2. **C3 — off-by-one** `range(toch-1)` в импедансах vs `range(toch)` в Mn/Nn.
   → `range(toch)`. (При toch ≥ Уискомба эффект мал, но формально верно.)

3. **Комплексная ε — потеря поглощения.** `sigma = sqrt(|ε|·|μ|)` (модуль) делал
   аргумент функций Бесселя вещественным → затухание внутри слоя терялось. Q_abs был
   занижен в 50–200× и даже отрицателен (нефизично); Q_sca ошибался на ~20% уже при
   tan δ ≈ 0.04. → `sigma = sqrt(ε·μ)` (комплексный показатель).

4. **Квадрант фазы для металлов.** `alfa = atan(Im/Re)` терял квадрант при Re(ε)<0
   (реальные металлы/плазмоника) → ~13% ошибки. → `alfa = phase(ε)` (atan2).

5. **Численная устойчивость (PEC-прокси).** Большая комплексная \|ε\| (например −1.7e7j
   как модель металла) вызывала переполнение Бесселя (e^{~6000}). → Экспоненциально
   масштабированные `jve/yve`: масштаб сокращается во всех отношениях и кросс-произведениях
   C/S, результат тот же, но без переполнения. Восстанавливает PEC-примеры репозитория.

## Подтверждено как НЕ баг (арбитр отклонил ложную правку)

- **Конвенция Ханкеля** (`hankel1`, e^{−iωt}): обратное рассеяние Q_b совпало с аналитикой
  → знаки и `hankel1` корректны; менять на `hankel2` НЕ нужно (исправлен лишь комментарий).
- **Сопряжение Mn/Nn**: не влияет на Q_sca/Q_ext/Q_abs (Re и |·|² инвариантны); безвредно.

## Важно для опубликованных результатов

Баги C4 и «потеря поглощения» сильнее всего влияют на **малые ka** и **поглощающие
материалы**. Многослойные линзы (Люнеберг/Максвелл, ~7 слоёв) и кейсы с потерями стоит
**перепрогнать на исправленном коде** и пересверить с HFSS.

## Осталось распространить

содержат **те же** баги (C3/C4/комплексная ε/квадрант/переполнение). Их планируется
перевести на единое верифицированное ядро `01_sphere` (фасад `sphere_core`), чтобы фикс распространился
автоматически и ушло дублирование.
