# Внешние эталоны для валидации / External validation references

## `examples.json` — эталонные диаграммы направленности Ansys HFSS

Наборы параметров слоистых сфер и соответствующие им **эталонные угловые
диаграммы направленности (ДН)**, рассчитанные в Ansys HFSS (метод конечных
элементов). Используются как **независимый FEM-эталон** для проверки
полной угловой диаграммы ядра слоистой сферы (`green_tensor/01_sphere.py`),
в дополнение к аналитическому ряду Ми (`tests/analytic_mie.py`).

### Содержимое
4 кейса слоистых сфер + 3 эталонных массива HFSS:

| Кейс | Слои (n) | a (радиусы) | eps[0] | k0 | Эталон |
|---|---|---|---|---|---|
| Линза Люнеберга | 5 | 0.2…1.0 | 1.96 (градиент) | 10π | `DN_NORM1` |
| Металл, 5 слоёв | 6 | 0.1…1.0 | 1e40 (PEC) | 4π | `DN_NORM4` |
| Металл, 2 слоя | 3 | 0.25,0.5,1 | 1e40 (PEC) | 4π | `DN_NORM4` |
| Металл + диэл. покрытие | 3 | 0.5,0.75,1 | 1e40 (PEC) | 4π | `DN_NORM5` |

Массивы `DN_NORM1/4/5` — нормированная ДН в дБ (пик = 0 дБ), по **359 точек**
(равномерная угловая сетка). Внешний слой каждого кейса — вакуум (eps=1):
это контракт API слоистой сферы.

### Формат (важно)
Файл — **последовательность конкатенированных JSON-объектов**, разделённых
пустыми строками, а НЕ единый JSON-документ. `json.load()` на нём упадёт
(«Extra data»). Парсить через цикл `JSONDecoder().raw_decode`:

```python
import json
dec = json.JSONDecoder()
raw = open("examples.json", encoding="utf-8").read()
objs, i = [], 0
while i < len(raw):
    while i < len(raw) and raw[i].isspace():
        i += 1
    if i >= len(raw):
        break
    obj, i = dec.raw_decode(raw, i)
    objs.append(obj)
```

Поле `k0` задано строкой-выражением (например `"10 * (math.pi)"`) — требует
вычисления при загрузке. Поле `DN_NORM_Ansys` ссылается на имя массива-эталона.

### Назначение
Регрессионный тест: посчитать ДН ядром `01_sphere` для каждого кейса и
сравнить с соответствующим `DN_NORM*` (числовая метрика, не визуально).
Закрывает класс O(1)-ошибок, незаметных при визуальном наложении графиков.
Регрессионный тест ещё не написан — это следующий шаг.
